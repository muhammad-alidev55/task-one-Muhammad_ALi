# Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML5** and **CSS3**. This project showcases a clean portfolio layout with smooth navigation, responsive design, and modern UI styling.

---

## Project Overview

The project focuses on writing clean HTML structure and modern CSS styling without using any JavaScript or frameworks.

---

## Features

- Responsive Layout
- Sticky Navigation Bar
- Hero Section
- About Section
- Projects Gallery
- Contact Section
- Modern Gradient Design
- CSS Grid Layout
- Hover Animations
- Mobile Responsive Design
- Clean and Organized Code

---

## Technologies Used

- HTML5
- CSS3
